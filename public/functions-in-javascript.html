Functions + Callback Functions

Understanding Functions and Callback Functions in JavaScript

In JavaScript, functions are fundamental blocks of code that can be defined and invoked to perform a specific task. They are reusable and can take parameters and return values. Callback functions, on the other hand, are functions passed as arguments to another function to be executed later, often after an operation completes.

Basic Function Syntax

Functions in JavaScript can be defined using the function keyword followed by a name (optional) and a set of parentheses containing zero or more parameters. The function body is enclosed in curly braces ({}).


        function add(a, b) {
            return a + b;
        }
        
        const result = add(5, 3);
        console.log(result); // Output: 8

Callback Functions

Callback functions are functions passed as arguments to another function and executed later within that function. They are often used in scenarios like event handling or asynchronous operations.

Here's a simple example of using a callback function:


        function greet(name, callback) {
            console.log('Hello, ' + name + '!');
            callback(); // Execute the callback function
        }
        
        function sayGoodbye() {
            console.log('Goodbye!');
        }
        
        // Using greet function with a callback
        greet('John', sayGoodbye);

In this example, greet is a function that takes a name parameter and a callback function. After greeting the person with the provided name, it executes the callback function. We pass sayGoodbye function as the callback, which simply logs 'Goodbye!' to the console.
